### Скрипт 1.1 описание:
Скрипт определяет средние символы слова: находит центральную букву для слов с нечётной длиной или две центральные буквы для слов с чётной длиной. Для этого вычисляется средний индекс делением длины строки пополам, после чего в зависимости от чётности длины извлекается либо один символ по среднему индексу, либо два смежных символа вокруг центра.

### Скрипт 1.2 описание:
Скрипт сортирует списки парней и девушек по алфавиту и выводит пары с одинаковыми индексами, а если длины списков не совпадают — печатает предупреждение, что кто‑то может остаться без пары.
